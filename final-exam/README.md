## 建議練習順序

1. 先啟動 MariaDB。
2. 用「原題簡化版」照題目順序練習 16 題。
3. 如果某一題看不懂，再回去查「完整詳解版」。
4. MariaDB 練完後，再練 MongoDB 匯入、查詢、更新、刪除。
